# Spanish Tax Obligations

## When to Use
Use this skill when the user:
- Asks about tax filing deadlines or obligations
- Needs to prepare any Spanish tax form (Modelo 303, 202, 200, 111, 190, 347, 390)
- Wants to calculate IVA, IS, or IRPF retention amounts
- Asks about deductible expenses
- Needs help with the annual corporate tax return (Modelo 200)
- Wants to understand fractional payments (pagos fraccionados)

## Workflow

### Para preparar un modelo trimestral (303, 111):
1. Preguntar por el trimestre y año
2. Solicitar datos de ventas/gastos del período (o pedir export de Holded)
3. Calcular paso a paso con desglose de casillas
4. Generar borrador con todas las casillas rellenadas
5. Indicar plazo de presentación y método (Sede Electrónica + certificado digital)

### Para preparar el IS anual (200):
1. Solicitar balance y PyG del ejercicio (de Holded)
2. Identificar ajustes extracontables
3. Aplicar incentivos: ERD, Ley Startups, I+D+i, reserva nivelación
4. Calcular liquidación completa
5. **RECOMENDAR revisión humana** antes de presentar

### Para consultas sobre deducibilidad:
1. Identificar el gasto concreto
2. Verificar contra Art. 15 LIS (gastos no deducibles)
3. Verificar condiciones específicas (vinculadas, límites, justificación)
4. Responder con artículo de ley + criterio interpretativo

## Critical Rules
- SIEMPRE citar el artículo de ley aplicable
- SIEMPRE mostrar el cálculo paso a paso
- Si hay duda interpretativa, adoptar posición conservadora y advertir
- Recordar plazos de presentación y consecuencias del retraso (recargo 1-20% + intereses)
- Para el Modelo 200: SIEMPRE recomendar revisión humana
